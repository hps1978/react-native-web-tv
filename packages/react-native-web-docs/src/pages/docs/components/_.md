Most component pages in this section describe the baseline React Native Web component behavior inherited by this package. Components that participate directly in TV focus and remote interaction, such as [View]({{ '/docs/view/' | url }}), [Pressable]({{ '/docs/pressable/' | url }}), [TextInput]({{ '/docs/text-input/' | url }}), [TVFocusGuideView]({{ '/docs/tv-focus-guide-view/' | url }}), and [TVTextScrollView]({{ '/docs/tv-text-scroll-view/' | url }}), include the additional props and behavior used for TV integration.
