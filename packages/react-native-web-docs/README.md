# React Native for Web documentation

Markdown pages and an Eleventy static site generator.

## Local development

From monorepo root:

```bash
npm run dev -w react-native-web-docs
```

Production build:

```bash
npm run build -w react-native-web-docs
```

Debug mode:

```bash
npm run debug -w react-native-web-docs
```
